Edit Permissions
chmod a+x 341fe

Run using
./341fe [flag] filename

 Flags:
-h = Shows a list of valid command-line arguments for the 341fe
-s = Reads the file passed and returns tokens from the scanner 
-p = Reads the file, scans and parses it and   builds the intermediate representation and then reports success or all errors it encountered .
-r = Reads the file, scans and parses it and display the intermediate representation  in readable form.
